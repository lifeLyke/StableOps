# StableOps

AI-powered tools for therapeutic riding programs (Emily). Python + Streamlit prototype.

## Quick start

See **[python_prototype/README.md](python_prototype/README.md)** for setup and run instructions.

```bash
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r python_prototype/requirements.txt
streamlit run python_prototype/app.py
```

This branch (`python_prototype`) contains the Python-only app. The Expo/React Native app was removed here.
